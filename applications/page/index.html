Why do you want to be part of it?
I think the Summer Coding School by Virtual Identity is a great opportunity for many people to learn knew things and enhance their knowledge in this field. What was the biggest challenge with a web project so far?
The biggest challenge with a web project so far was to create a responsive website. The more smaller the devices get the less display area is available for the designer. Depending on the design, it can force the developers to plan a completely new design.
